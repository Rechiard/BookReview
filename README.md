# 万卷书堂项目

该项目Python爬虫为数据集，用SpringBoot+Shiro的框架设计搭建了书籍评价系统。采用shiro做安全登录验证，即安全也强大。

日常生活中用户对电子书的需求与日俱增，该项目对网络中大部分书籍进行收集和归纳，便于用户对电子书以及实体书进行收藏和评价，具体作用类似豆瓣图书。
